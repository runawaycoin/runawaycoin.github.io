    RC depit

    I used standard and common cPanel PHP shared hosting to host my Symfony based websites, which does not offer SSH access therefore I couldn't run the symfony console deployment routines on the server easily.

    So I developed my own automated deployment application, written in pure PHP.

    Some of its features are:

    • Runs yarn and encore / webpack to build client side assets before deploying, if needed
    • Compares files, and zips up any changed folders
    • FTPs the archived files to the server
    • Unzips the files on the server
    • Warms up the symfony cache on the server
    • Tags Git marking the release
    • Updates the database schema, only if needed
    • Tech:
      Git, Zip, FTP

    • Languages & Frameworks:
      PHP

    • Platform:
      Web

    • Date:
