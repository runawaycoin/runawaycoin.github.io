    rc-depit

    I used standard cPanel PHP shared hosting to host my Symfony based websites, which does not offer SSH access therefore i couldn't run the symfony console deployment routines on the server.

    Therefore i developed my own deployment routine application, written in PHP it:

    • Runs yarn and encore / webpack if needed
    • Compares the files, and zips up any changed folders
    • FTPs the zips to the server
    • unzips the files on the server
    • Warms up the symfony cache on the server
    • Tags Git marking the release
    • Updates the database schema if needed - backing the database up into a passworded zip
    • Features: Git, Zip, FTP
    • Platform: Web
    • Languages & Frameworks: PHP
    • Date: Oct 2017 - ongoing
