food-rating

About it - released as WP plugin

Easily display the UK Food Agency hygiene rating on wordpress pages.

This plugin is developed in PHP, using the WordPress API and connects to the Food Agency (ratings.food.gov.uk) JSON API to get the hygiene rating for a business and displays the correct rating banner.

There are several settings options to tweak the output of the plugin (banner size & text, default location to restrict the business search), this is done via the WordPress options API.

  • Features: JSON server side connection to Food Agency API, Word Press plugin
  • Platform: Web, Word Press plugin
  • Languages & Frameworks: PHP, MySQL, WordPress plugin SDK
  • Date: Dec 2016
